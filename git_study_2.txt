add //セーブしたい内容を追加
add . //変更した全てのファイルを追加する
commit //セーブポイント
commit -m "コメント" //コメント
push //アップロード

パソコンでの流れ
1.デスクトップなどどこでもいいのでフォルダを作る
2.gitのローカルリポジトリ(貯蔵庫)をつくる
git init (初期)
3.コード書く
4.add
5.commit

githubでリポジトリを作成
git branch -M main //mainブランチに変更
//ブランチはゲームのセーブデータのようなもの

どのremoteに送るのか設定
git remote add origin https://github.com/taka01150810/git_study.git

git remote -v
//結果
origin	https://github.com/taka01150810/git_study.git (fetch)
origin	https://github.com/taka01150810/git_study.git (push)

git push -u origin main //mainはブランチ
