リモートからダウンロード
大きく3種類
git clone 丸ごとダウンロード
git pull 差分ダウンロードしてマージ
git fetch 差分確認

git clone 丸ごとダウンロード
->デフォルトブランチをダウンロード
git clone <gitHubのurl>

git clone ブランチ指定
git clone -b <ブランチ名> <gitHubのurl>

git pull (fetch + merge) -> 差分ダウンロード + マージ
git pull origin <ブランチ名>

git pull のwarning
Git 2.27.0で導入されたメッセージ
rebase false //デフォルトの挙動
rebase true //--rebase実行(コミット履歴一直線)
ff only //fast-forward
