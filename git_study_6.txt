取り消したい内容の種類
addする前に書いたコード
git add
git commit
git push
pull request
merge pull request
git merge
commit コメントの修正

取り消し系のコマンド
git clean
git reset
git revert
git restore
(以前は git checkoutだったけれどswitchとrestoreに分離)
それぞれ複数のオプションがあります

addしてないコードを消したい
->VSCodeのソース管理からでも消せる
->簡単

新規作成ファイルを消す
git clean -f <ファイル名>

ファイル変更を取り消す
git restore <フォルダ/ファイル> //ファイル取消
git restore . //add全て取消 -> リモート追跡リポジトリとの比較
