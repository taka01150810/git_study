GitHub Pages
->静的サイトのホスティングサービス
(HTML・CSS・JS)

内部でJekyll(ジキル)を使い構築している
独自ドメインを設定することもできる

index.html、index.md、README.md の順で表示
注意事項
①privateリポジトリの場合Proアカウントが必要
②商用利用不可
③1GBの推奨上限

手順
index.htmlなどで書く -> Setting -> Pages -> Save
結果 URLが生成され、ページが作れる。

git cherry-pick
さくらんぼのように熟れた実だけを取る
->欲しいコミットだけ選んでマージする

コマンド
git cherry-pick コミットID // 1件
git cherry-pick コミットID コミットID //複数同時

オプション
--skip //現在のコミットをスキップする
--continue //チェリーピックを続行
--quit //チェリーピックを止める
--abort //チェリーピックをなかった事にする

git rebase
commit履歴をキレイにする
->①2つのcommitの履歴を一直線にする

git rebase ブランチ名

->②複数のcommitを1つのcommitにまとめる
